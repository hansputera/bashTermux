# Free Source bash Termux

## Instalation

- `pkg install git -y`
- `pkg install nano -y`
- `git clone https://github.com/hansputera/bashTermux.git`
- `cd bashTermux`
- `mv source.sh ../../usr/etc`
- `cd ../../usr/etc`
- `rm -rf bash.bashrc` 
- `nano source.sh`
- `(Beri satu spasi untuk mengganti nama file`
- `di konfigurasi ganti nama ganti nama jadi bash.bashrc`
